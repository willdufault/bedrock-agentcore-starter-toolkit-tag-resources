# bedrock-agentcore-starter-toolkit-tag-resources

[bedrock-agentcore-starter-toolkit #136: Feature request: Tag all resources created by runtime launch](https://github.com/aws/bedrock-agentcore-starter-toolkit/issues/136)

This repo contains a Python script to tag the following resources created by the AgentCore Starter Toolkit (0.3.2):
- S3 bucket for storing CodeBuild sources
- ECR repo for storing image
- CodeBuild project for building image
- AgentCore runtime
- `DEFAULT` AgentCore runtime endpoint
- AgentCore memory for the agent
